# Smart-vending-machine

### 資工1B 第十二組
### *蔡承奎 童冠晨 黃彥慈
### Program introduction
#####這段程式是一個簡易的智慧販賣機
#####目前也沒有能力使用資料庫做後台
#####一開始的註冊帳密，可以保證密碼不被外洩
#####使用者可以嘗試不同試錯，系統會有回應喔

#####每個段落運用清屏的功能，維持版面整齊
#####必要時利用'sleep_for'時間函數停頓
### Rule of procedure
#####首先註冊帳密時，不能直接點擊回車鍵
#####字數也有限制，某些情況會直接跳出程序

### Procedureal gameplay
#####註冊帳密
#####輸入帳密
#####選擇語言
#####選擇商品
